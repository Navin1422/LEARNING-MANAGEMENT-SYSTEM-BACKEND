# Learning Management System - Backend

A Node.js backend API for a Learning Management System (LMS) built with Express.js.

## Environment variables

Create a `.env` file at the project root (do not commit it). You can copy the provided `.env.example` and fill in your credentials.

Required variables:
- `MONGO_URL` — your MongoDB connection string (e.g., from MongoDB Atlas).
- `PORT` — optional port for the server (defaults to `21000`).

The repository already lists `.env` in `.gitignore` to avoid leaking secrets.
